# Imágenes para AriMar FoodLab

Este directorio contiene todas las imágenes locales utilizadas en la web.

## Estructura de imágenes requeridas

### Home (/)
- `hero-variedad-tuppers.webp` - Imagen de fondo del hero
- `carta-crema.webp` - Bloque "Hoy en cocina"
- `carta-potaje.webp` - Bloque "Hoy en cocina"
- `carta-pasta.webp` - Bloque "Hoy en cocina"
- `cocina-lote-linea.webp` - Bloque "Cómo cocinamos"
- `residuo0-caducidad.webp` - Bloque "Cocinar sin desperdiciar"
- `residuo0-aprovechamiento.webp` - Bloque "Cocinar sin desperdiciar"
- `packaging-tupper.webp` - Bloque "Envases y presentación"

### Carta (/carta)
- `carta-crema.webp`
- `carta-potaje.webp`
- `carta-pasta.webp`
- `carta-carne-estofada.webp`
- `carta-vegetal.webp`
- `carta-postre-vaso.webp`

### Nuestra Cocina (/nuestra-cocina)
- `cocina-emplatado.webp` - Imagen cabecera
- `cocina-servicio.webp` - Sección "Producción real"

### Pedir (/pedir)
- `hero-variedad-tuppers.webp` - Imagen de fondo

### Fondo global
- `bg-textura-reciclado.webp` - Textura sutil para fondo (opacidad 5-8%, repetible)

## Notas
- Todas las imágenes deben estar en formato WEBP
- Si una imagen no existe, se mostrará un placeholder neutro
- Las imágenes se optimizan automáticamente con Next.js Image
- Se aplica lazy loading para mejorar el rendimiento
